🤖 Gemini AI Code Review Reports

Access your comprehensive code analysis reports in multiple formats

📍 Where to Find Reports

Reports are generated in 3 locations:

  1. 📁 File System: reports/ directory (JSON, HTML, Markdown)
  2. 💬 GitHub PR Comments: Enhanced review comments with summaries
  3. 🔍 GitHub Actions Logs: Detailed analysis logs in Actions tab

📊 Available Report Formats

📝 Markdown Report
GitHub-friendly format with issue breakdown and action items
test-report.md
📊 JSON Report
Structured data for programmatic access and API integration
test-report.json
🌐 HTML Report
Beautiful web report with charts and visualizations
Generated during PR review

🧪 Test Report Generation

To generate test reports right now:

# Test the rule engine and generate sample reports
node tests/test-rule-engine.js

# Test individual components
npm run test-rules
npm run test-simple

🚀 Live PR Reports

When you create a real PR, reports will be generated at:

reports/
├── pr-{number}-report.json # 📊 API data
├── pr-{number}-report.html # 🌐 Web report
└── pr-{number}-report.md # 📝 GitHub format

Plus enhanced PR comments with:

  • Overall score and status
  • Security analysis with compliance status
  • Issue breakdown by severity and category
  • Must-fix items with line numbers and suggestions
  • Links to detailed reports
  • Technical debt estimation
